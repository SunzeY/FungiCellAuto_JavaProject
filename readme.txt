本项目是2021年美国大学生数学竞赛A题的fungi的元胞自动机的模拟程序（java编写），基本只具有观赏性！
exe文件包含了一个无需java环境的独立程序。
1.jre是.exe运行所需的环境，本机如果没配置java到%HOME%中就不要
将.exe和jre文件夹分开（保证在同一个文件夹下）
2.目前实验的data_out.txt由于存成数组的形式，可以被python直接读取，用于
绘图（java绘图操作是在过于复杂）
3.下面两个框中左边的框返回一定的错误信息（还有部分比较复杂的错误信息保留在
console输出），右边没用
4.由于右上角的数据输入框本质是一个二维数组，请保证读取数据的时候，被选定的输入
块不需要被读入数据（否则你占着输入肯定是没法读取）
5.run按钮本质时开辟一个新的线程，将主窗口的状态复制到新的窗口，结束后不能关闭窗口
（关了的话整个程序也会关闭）
